# HR-Analytics-Dashboard-PowerBI
## Overview
This project is an interactive HR dashboard built using Power BI to analyze employee attrition and workforce trends.

## Key Insights
- Sales Executive has the highest attrition rate
- Highest attrition observed in employees aged 25–34, especially males
- Life Sciences and Medical graduates show higher turnover

## Features
- KPI cards for employee metrics
- Department-wise attrition analysis
- Job role and job satisfaction insights
- Age, gender, and education-based filtering

## Tools Used
- Power BI
- Data Visualization
- DAX
